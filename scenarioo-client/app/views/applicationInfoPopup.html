About Scenarioo

Documentation is in the Code.

Our vision

Scenarioo is a user scenario documentation, a functional documentation of the software describing all use cases and the most important user scenarios of your software. Each scenario is documented by step by step screenshots and a lot of additional meta information about what is going on in your application.

With the help of webtests and Scenarioo the documentation of your application will never be out of date. You can browse through the nightly created screenshots or compare the newest version to the version of the last release.

You can use shortcuts to navigate through the steps of a scenario.

Scenarioo is an education project by some developers from Switzerland to develop a web application that can be used to browse "User Scenario Documentations", which are documentations generated automatically from your UI tests (e.g. by Selenium).
